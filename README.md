# Open Decision Framework
This is a remix of the community version of the [Open Decision Framework](https://github.com/opensourceway/open-decision-framework)

## Background
Red Hat put together and published an Open Decision Framework, a collection of best practices for applying [open source principles](https://opensource.com/open-source-way) to business projects and decisions. 

This remix is designed with a specific project in mind, the development of Planet 4 a Wordpress based Content Management System for Greenpeace. This new version contains a practical mapping of communication tools and techniques to help open decision making.
